出雲見月是一名 Non-binary 大學生，長期壓抑的它因為得不到父母和學校的理解，而逐漸變得孤僻。
某一天，在朋友的幫助下，出雲終於決定走出房間，到位於新宿的跨性別友好酒吧 Shelter 打工。
於是在那裡遇見了各種各樣的客人和同事，可正當自己的調酒師生涯剛剛開始之時，因為突入其來的傳染病，
使得酒吧陷入了閉店的危機，這一次輪到少女們拯救 Shelter 了，她們可以共同渡過難關嗎？
